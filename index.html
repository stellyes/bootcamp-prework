HTML

  • <!DOCTYPE html> is a declaration that tells the browser what kind of document to expect to load.
  • The <html> tag is used to initialize the HTML document, often paired with a lang="en" attribute to establish the webpage's language. In this case, the language would be English.
  • The head tag is used to establish webpage metadata, with child tags including everything from <meta> to <title>.
  • The body tag is where the content of the webpage goes.
  • Within the body tag, child tags like section and div are used to section off different sections of the webpage to create a sense of organization and flow.

CSS

  • Inline CSS is when an element is styled directly in the HTML tag.
  • You can also style all HTML tags of the same variety by using <style>, putting the HTML tag you want to modify followed by {} brackets, and the attribute you wish to modify within those brackets.
  • Most often, we want to use external CSS style sheets, which can be used with the <link> tag.
  • A margin indicates how much space we want around the outside of the HTML element
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • When you want to include a JavaScript section in your HTML file, use the <script> tag with the src attribute to identify where the .js file is located.
  • The boolean operator to compare equality in JS is ===
  • console.log(); is used to log specific data to print to the console. This can prove useful for debugging.
  • Variables are declared using var, followed by the variable name, an '=' sign, and the value. JavaScript evaluates variable types at runtime, so there's no need to declare the specific type inline.
  • Functions are declared using function followed by the functionName(), and {} brackets to encase it.
  • Math.floor() and Math.random() are both properties of JavaScript's built-in Math object that can be used together to generate a random number. To learn more about the Math object properties and methods, visit the MDN Web Docs on Math
