HTML

  • <!DOCTYPE html> is a declaration that tells the browser what kind of document to expect to load.
  • The <html> tag is used to initialize the HTML document, often paired with a lang="en" attribute to establish the webpage's language. In this case, the language would be English.
  • The head tag is used to establish webpage metadata, with child tags including everything from <meta> to <title>.
  • The body tag is where the content of the webpage goes.
  • Within the body tag, child tags like section and div are used to section off different sections of the webpage to create a sense of organization and flow.

CSS

  • Inline CSS is when an element is styled directly in the HTML tag.
  • You can also style all HTML tags of the same variety by using <style>, putting the HTML tag you want to modify followed by {} brackets, and the attribute you wish to modify within those brackets.
  • Most often, we want to use external CSS style sheets, which can be used with the <link> tag.
  • A margin indicates how much space we want around the outside of the HTML element
  • A padding indicates how much space we want around the content inside an element.
